# 타입 확장

```ts
import type { BlockType } from 'ddo-dnd';

type MyBlock = BlockType & {
  title: string;
  color?: string;
};
```

## 제네릭 훅

```ts
useDragBlock<MyBlock>({ workBlocks, updateWorkBlocks, ... })
useRotateBlock<MyBlock>({ workBlocks, updateWorkBlocks, ... })
useBlocksTransition<MyBlock>(updateWorkBlocks)
```
